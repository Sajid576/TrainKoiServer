(init npm with default config of package.json)
npm init -y  

(install firebase dependencies in that project folder)
>>npm install --save firebase	

(install firebase tools globally in your working PC)
>>npm install -g firebase-tools	

//in cmd to check if it is installed and its version:
>>npm firebase -version

(below procedures are for running a local web server to communicate with firebase)

>>firebase init 

>>firebase serve

*******
To bring back the node modules
>> npm i
